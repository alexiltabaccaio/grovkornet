🚀 Release: Version 0.12.0 (Beta Update) @everyone - A massive update is rolling out for testers!

This release brings new effects, major performance boosts, and a huge number of bug fixes!

✨ What's New & Improved:

🔹 New Effects: Added Video Stabilization, Asymmetric Chroma Bleed, Analog Halation, and a new Optical Effects module! Redesigned tape jitter shader and disabled adaptive noise reduction by default to preserve the analog feel.
🔹 Performance: Optimized frame rendering and UI controls to eliminate delays. Added instant double-tap zoom in the gallery.
🔹 Features: Added opt-in screen capture security and improved the theme selector.
🔹 Open Source: The repository is officially relicensed under GPLv3!

🐛 Bug Fixes:

🔸 Camera & Memory: Fixed memory leaks and crashes during image processing. Fixed 30fps lock on front camera and flash conflicts.
🔸 UI & Gestures: Prevented UI thread crashes. Fixed stale slider states, input lag, and touch leaks.
🔸 Effects: Fixed mismatches (rotation, scale, orientation) between preview and capture. Fixed stretched edges and aligned high-res effects.
🔸 Stability: Fixed major issues when returning from the background (blank previews, battery drain, frozen drawers). Resolved gallery navigation glitches.
🔸 Navigation: Improved Android back button handling, fixed disappearing footers and viewfinder offsets.

Keep the feedback coming! Drop bugs in #bug-report 🛠️ or join #general 💬.

Have fun shooting! 📸
